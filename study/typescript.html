The TypeScript Compiler (and its Configuration)

Create tsconfig.json

The tsconfig.json file corresponds to the configuration of the TypeScript compiler (tsc).

$ tsc --init

            "compilerOptions": {
            "target": "es6",

target: ECMAScript target version (defualt es5)
es6: using let, const, typescript arrow function.
module:
sourceMap: generates corresponding ‘.map’ file so that it can be used in the production code for debugging.
outDir: where to locate the build file.
rootDir: root directory of input files.

# Using number, boolean and string types.

                                
            function cal(num1: number, num2: number, printResult: boolean, showResult: string){
                if(typeof num1 === 'number' && printResult === true){
                    return num1 * num2;
                } else {
                    throw new Error('Wrong type');
                }
            }
            const num1 = 2;
            const num2 = 30.5;
            const printResult = true;
            const showResult = 'result:'
            const result = cal(num1, num2, printResult, showResult);

# Using Union type.

        
                function Animals(dog: number | string, cat: number | string) {
                    let total;
                    if(typeof dog === 'number' && typeof cat === 'number'){
                       total =  dog + cat;
                    } else {
                        total = dog .toString() + cat.toString();
                    }
                    return total;
                } 
                
                const animalTotal = Animals(100, 16);
                console.log(animalTotal);
                
                const animalName = Animals('Johnny Hook', 'Johnny Jim');
                console.log(animalName);

**Note: toString () method converts a number to a string.

Function Types & Callbacks

TypeScript supports callback functions to make your program asynchronous, a function which is scheduled to be called after some asynchronous processing is completed. 
                    
                    function addAndHandle(n1:number, n2: number, cb: (num: number) => void){
                        const result = n1 + n2;
                         cb(result);
                    }
                    
                    
                    addAndHandle(10,20, (result) => {
                        console.log(result);
                    })
